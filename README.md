## Adam Bailey Blog/Portfolio
This application is built with [VitePress](https://vitepress.dev/), a vue powered static site generator.
### Build and test locally
`npm run blog:dev`

### Deployment
`npm run blog:build`

point to `blog/.vitepress/dist` in your web server
